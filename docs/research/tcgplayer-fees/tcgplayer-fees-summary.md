# TCGPlayer Fee Structure (Official)
> Source: https://help.tcgplayer.com/hc/en-us/articles/201357836-TCGplayer-Fees
> Source: https://help.tcgplayer.com/hc/en-us/articles/360047732673-Fee-Calculation-Examples

## Fee Types

- **Marketplace Commission Fee** — Percentage fee sellers pay on each sale
- **Pro Fee** — Additional commission for Pro Tools users (N/A for standard sellers)
- **Direct Shipping Replacement Cost (SRC)** — Fee on Direct orders only (N/A for standard marketplace sellers)
- **Sync Fee** — Fee for third-party platform sellers (Crystal Commerce, BinderPOS, Ion Sync)
- **Transaction Fee** — Fixed per-transaction fee (2.5% + $0.30). A "transaction" = any number of items purchased by the same buyer from the same seller at checkout using the same shipping method.

## Fee Calculation (Level 1-4 Standard — Dustin's tier)

**Total Cost = Marketplace Commission + Transaction Fee**

- Marketplace Commission = order subtotal × 10.75% (capped at $75 per product)
- Transaction Fee = (order total [items + shipping + tax] × 2.5%) + $0.30

**Combined effective rate: ~13.25% + $0.30 per order**

## Key Notes
- Transaction fee is per ORDER, not per card — consolidation helps
- Marketplace commission is capped at $75 per product sold
- International (non-US) sellers on PayPal: additional 2% processing fee (capped at $20)
- All fees apply to the order, not individual line items
